# 🚀 Frontend UI Library

A collection of modern, responsive, and production-ready UI components built with **HTML**, **CSS**, and **JavaScript**.

Whether you're a beginner or an experienced frontend developer, you'll find reusable UI components, animations, and interactive elements to speed up your workflow.

---

## ✨ Features

- Modern UI Components
- Responsive Design
- Clean & Well-Organized Code
- HTML, CSS & JavaScript Only
- Easy to Customize
- Beginner Friendly
- Production-Ready Examples

---

## 📂 Components

- 🔘 Buttons
- 🔍 Search Bars
- 🔐 OTP Inputs
- ☑️ Checkboxes
- 🎚️ Toggle Switches
- 📋 Dropdowns
- 💬 Tooltips
- 🔔 Toast Notifications
- 📊 Progress Bars
- 🃏 Cards
- 📱 Navigation Menus
- 🎨 Hover Effects
- ✨ CSS Animations

> New UI components are added regularly.

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/DesignCodeWithAV/frontend-ui-library.git
```

2. Open any component folder.

3. Run `index.html` in your browser.
